Tubular Grids with .NET assembly

Tubular provides a .NET library to easily create responses in WebApi (or WCF) for your Tubular grids. This helpers and extension methods provide sorting, filtering and pagination using System.Linq.Dynamic.

You first need to setup your ApiController. All the request made by Tubular Grid are using HTTP POST method, in order to send a GridDataRequest model. This model provides an extension method to generate a payload using any IQueryable data source like an EntityFramework query.

The internal helper will deliver a GridDataResponse model. This model is the expected payload for Tubular Grid.

Heads up! You can also use this helper with EmbedIO web server.
[RoutePrefix("api/users")] public class UsersController : ApiController { [HttpPost, Route("paged")] public IHttpActionResult GridData([FromBody] GridDataRequest request) { using (var context = new SampleDbContext(false)) { return Ok(request.CreateGridDataResponse(context.SystemUsers)); } } }

Since you pass a IQueryable as parameter, you can filter or apply any security level control before to call the CreateGridDataResponse.
